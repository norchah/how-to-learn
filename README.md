# Первая проектная работа. "Научиться учиться". #  

## Знакомство с `html` и `CSS` ##
  Проектная работа "Научиться учиться" создавалась для закрепления теории и работы в тренажере, наработки следующих навыков в `html` и `CSS`:
  * Семантичная разметка документа;
  * Позиционирование;
  * Оступы и размеры блоков, элементов;
  * Создание сеток и понятие потока документа;
  * Анимации и трансформации, плавные переходы;  
  Сама работа так же учит, как правильно усваивать информацию, разным техникам получения и удержания информации в голове.  
  
  Использовались классические техники начального уровня языков разметки.
